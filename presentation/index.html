What Sets Us Apart
Innovation Built Into the Core,
Not Bolted On
🗣️
Voice-First, Not Text-First with Voice Added
Designed from day one for students who speak before they type. Mic is the primary input, keyboard is the fallback — not the other way around.
🪴
Local-Analogy Mode (Not Just Translation)
Tuition-style mode rewrites explanations using cricket, kirana shops, and farming — the same mental models a good local tutor uses. Not a translation wrapper.
🛡️
Responsible AI: Confidence Calibration
Every answer shows a confidence badge. Low-confidence answers explicitly tell students to verify with their teacher. No hallucination is presented as fact.
📡
Low-Bandwidth Mode for Tier-2/3 Reality
One toggle disables audio autoplay and requests compressed responses. Designed for 2G connections and budget Android devices, not Jio-fiber Bangalore.
Scalability & Impact

Built to Scale Across Bharat

🌐 Language Scalability
  • Language-pack architecture — adding a new Indian language is a config change, not a rebuild
  • Hindi, Tamil, Bengali wired & tested; Marathi, Telugu, Kannada next
  • Web Speech API coverage growing — or swap to Sarvam/AI4Bharat TTS for production
  • Script-native rendering for Devanagari, Tamil, Bengali — no Latin-script fallback
⚙️ Technical Scalability
  • Stateless FastAPI backend — horizontally scalable behind a load balancer
  • SQLite swappable for PostgreSQL with one env-var change
  • Mock data ↔ OpenAI API toggle without any code change
  • PWA-capable frontend — installable on Android home screen
  • Server-Sent Events for streaming — works on HTTP/1.1, no WebSocket required
